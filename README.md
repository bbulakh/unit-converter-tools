# Cools

A collection of unit-converter tools. Built with Astro and Vue.

## Getting started

1. Clone the project
```
git clone https://github.com/bbulakh/unit-converter-tools.git
```

2. Install the dependencies:
```
npm install
```

3. Run the project
```
# Start server
npm run start

# or Build 
npm run build
```
